# Card Game 24

Welcome to the Card Game 24! This project is a fun and engaging card game that challenges players to use basic arithmetic operations to reach the number 24 using four cards. The game can be played in Solo or Multiplayer mode.

## Project Structure

The project is organized as follows:

```
card-game-24
├── src
│   ├── index.html         # Main HTML document for the game
│   ├── js
│   │   ├── game.js        # Main game logic
│   │   ├── ui.js          # User interface management
│   │   └── utils.js       # Utility functions
│   ├── css
│   │   ├── styles.css     # General styles
│   │   └── poker-table.css # Poker table specific styles
│   └── assets
│       └── cards          # Directory for card images
├── package.json           # npm configuration file
└── README.md              # Project documentation
```

## Setup Instructions

1. **Clone the repository:**
   ```
   git clone <repository-url>
   cd card-game-24
   ```

2. **Install dependencies:**
   ```
   npm install
   ```

3. **Open the game:**
   Open `src/index.html` in your web browser to start playing the game.

## Game Rules

- The objective of the game is to use four cards and basic arithmetic operations (addition, subtraction, multiplication, and division) to reach the number 24.
- Players can play in Solo mode or challenge their friends in Multiplayer mode.
- Each player takes turns to make their calculations using the cards provided.

## How to Play

1. Choose between Solo or Multiplayer mode on the title page.
2. In each round, four cards will be displayed.
3. Use the cards and arithmetic operations to calculate the number 24.
4. Submit your answer to see if you are correct!

Enjoy the game and may the best player win!
